# Pharmacology 101 Study Guide

An interactive, mobile-first study guide for medical and pharmacology students, featuring comprehensive notes, real-time quizzes, and custom visual learning aids.

[![Live Site](https://img.shields.io/badge/Live%20Site-Click%20Here-blue?style=for-the-badge&logoColor=white)](http://mustafa-shah-tech.github.io/Pharmacology_Study_Guide/) <!-- live link-->

## Features
- **Mobile First & Fully Responsive:** Works flawlessly on any display size (desktop, tablet, and mobile) with an app-like hamburger menu sidebar toggle.
- **Red Exam Alerts:** Visual components highlighting high-probability questions and crucial exam concepts.
- **Interactive Min-Quizzes:** Features built-in checkpoints with immediate grading and detailed feedback.
- **Local Storage Saves:** Automatically saves your completed progress locally so you never have to start over.
- **Rich Visuals:** Includes comprehensive comparison grids, timelines (e.g., drug development phases), and cleanly formatted formula boxes.
- **Custom UI Mode:** A pleasing dark theme optimized for long hours of comfortable studying.

## Technologies Used
- HTML5
- Vanilla Custom CSS (using variables and modern responsive grid/flexbox)
- Vanilla Javascript (`localStorage`, Intersection Observer API)

## Setup & Local Development
Since this is a vanilla HTML/CSS/JS application:
1. Simply download or clone the repository.
2. Open `index.html` in your web browser of choice.
